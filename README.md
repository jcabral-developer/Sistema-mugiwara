Sistema Mugiwara – Gestión Gastronómica

Sistema Mugiwara es una aplicación web desarrollada en PHP y MySQL, pensada como base para la gestión interna de negocios gastronómicos.
El proyecto se encuentra en desarrollo activo y forma parte de mis proyectos personales y freelance, donde adapto la funcionalidad según las necesidades de cada cliente.

Actualmente el sistema cuenta con un módulo de autenticación de usuarios, manejo de sesiones y una estructura preparada para seguir creciendo con nuevos módulos.
